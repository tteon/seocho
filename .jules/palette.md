## UX and Accessibility Improvements
- Refactored the `Reset Trace` button inline CSS into a reusable `.reset-btn` class with hover states.
- Added explicit `:focus-visible` outlines to all primary UI buttons (`.rail-btn`, `.reset-btn`, `#ingestBtn`, `#oneClickDemoBtn`, and `.composer button`) to improve keyboard navigation accessibility.
- Added explicit `aria-label` attributes to the vertical `.rail-btn` navigation buttons to provide clearer context for screen readers.
