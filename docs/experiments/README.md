# Experiments

This directory stores experiment notes, live-evidence reports, and
domain-specific validation material.

Experiment documents are not the first-read path for SEOCHO users. Promote a
finding into `docs/`, `README.md`, or an ADR only when it becomes an active
product or architecture contract.

Keep performance and scalability claims tied to the live-evidence rule:
identify service versions, dataset, concurrency, hardware or container limits,
warmup, and skipped components.
