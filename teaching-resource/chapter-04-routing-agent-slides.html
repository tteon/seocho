4.3 RRF — score scale 무시, rank만

def reciprocal_rank_fusion(per_backend, k=60, top_n=10):
    scores = defaultdict(float)
    for backend, ranked in per_backend.items():
        for rank, item in enumerate(ranked, start=1):
            scores[item['doc_id']] += 1.0 / (k + rank)
    return sorted(scores.items(), key=lambda kv: kv[1], reverse=True)[:top_n]
vector cosine 과 BM25 raw score 합산은 위험. RRF는 rank만 보므로 robust.

인용 검증 — invalid count = fabrication 신호

CITE_RE = re.compile(r'\[src:([^,\]]+),\s*chunk:(\d+)\]')

def validate_citations(answer, context):
    found = set(CITE_RE.findall(answer))
    valid = set(CITE_RE.findall(context))
    return {'invalid': sorted(found - valid),
            'all_valid': found.issubset(valid)}
invalid 카운트가 0이 아니면 — Ch 5의 self-reflect 또는 debate가 필요한 시그널.
